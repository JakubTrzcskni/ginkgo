bool \mbox{\hyperlink{classgko_1_1solver_1_1BatchIdr}{gko\+::solver\+::\+Batch\+Idr}}$<$ Value\+Type $>$\+::parameters\+\_\+type\+::deterministic \{ false \}\hspace{0.3cm}{\ttfamily [mutable]}}



If set to true, the vectors spanning the subspace S are chosen deterministically. 

This is mostly needed for testing purposes.

Note\+: If \textquotesingle{}deterministic\textquotesingle{} is set to true, the subspace vectors are generated in serial on the C\+PU, which can be very slow.

The default behaviour is to choose the subspace vectors randomly.
