Detailed Description

template<typename ValueType>
class gko::EnableBatchScaledSolver< ValueType >

Provides basic functionality for solvers which need to be composed with batch scaling.

If A is a matrix, S is a batch-scaling operation and F is a solver-type such that F(A) is a solver, then if the class for F inherits from this EnableBatchScaledSolver class, it represents a new solver F(S(A)) (F composed with S applied to A).
